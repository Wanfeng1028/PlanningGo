# PlanningGo / 周末有谱

本项目是「周末有谱」本地生活活动规划 Agent 的前端实现。当前版本优先完成前端可上线演示体验，基于上传设计稿中的 43 页、28 页、最终 100 页与移动端 60 页重新归并为一套连续产品流程。

## 页面结构

桌面端主导航固定为：

```text
首页 / 功能 / 场景案例 / 设计亮点 / 开发者 / 个人中心
```

右侧固定注册和登录入口。适合弹窗化的身份选择、定位授权、预约、票务锁单、支付交接、多人投票、隐私导出、API Key 创建等流程已作为 Modal / 移动端 Bottom Sheet 处理。

移动端使用：

```text
首页 / 规划 / 地图 / 消息 / 我的
```

## 产品页面

设计稿不会按图片编号机械生成页面，而是归并成真实软件模块：

- 首页：价值说明、Demo 主线、路线地图、能力总览
- 功能：需求对话、定位约束、Agent 工作台、三方案对比、路线预订、授权执行、分享协作、What-if、记忆沉淀
- 场景案例：家庭、朋友、情侣、雨天、亲子低负担、晚出发压缩
- 设计亮点：设计系统、组件、状态、导航、数据矩阵
- 开发者：API Key、工具日志、Webhook、质量看板、测试数据
- 个人中心：登录注册、画像、通知、隐私、记忆导出

## 技术栈

- React
- Vite
- TypeScript
- SCSS + CSS Modules
- Framer Motion
- lucide-react

本项目明确不使用 Tailwind CSS。

## 本地运行

```bash

npm run dev
```
