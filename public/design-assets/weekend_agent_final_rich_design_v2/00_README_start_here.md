## 核心 Demo 主线

小明从 **浙大紫金港** 出发，下午想去 **杭州西湖** 附近玩几个小时：

1. 一句话输入需求。
2. 定位授权或手动输入起点。
3. Agent 生成 A/B/C 三套方案。
4. 用户选择亲子低负担方案。
5. 地图展示：浙大紫金港 → 西湖断桥/白堤 → 湖滨休息 → 海底捞 → 回家。
6. 用户授权 Agent 执行预约/锁单/日历/分享。
7. Agent 预约海底捞、可选电影锁座、可选鲜花蛋糕配送。
8. 支付必须由用户本人确认。
9. 计划发给老婆或朋友投票。
10. 行程结束后评分并沉淀记忆。
