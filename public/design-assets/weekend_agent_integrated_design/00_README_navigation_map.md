登录、注册、游客画像、个人画像、通知、隐私、安全、账号等页面统一归入「个人中心」导航逻辑；对话收集、Agent 工作台、结果总览、方案对比、路线预订、授权、执行和协作页面统一归入「功能页卡」导航逻辑。

## 保留规则

- 未删除两份原始设计中的页面主题。
- 同名或近似页面保留两个版本：`D43_*_v1_refined_bold_nav.svg` 与 `I28_*_v2_desktop_nav_variant.svg`。
- D43 版本保留原导航并增强字体粗细。
- I28 版本保留原页面主体差异，裁切旧导航后套用 D43 风格导航。
